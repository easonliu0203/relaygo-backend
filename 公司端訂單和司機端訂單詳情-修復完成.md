## 📋 問題總結

**預期行為**:

- 應該顯示司機的完整姓名（例如：「小花 李」）

### 問題 2: 公司端訂單詳情無法顯示客戶和司機姓名

**症狀**:
- 訂單詳情頁面中的「客戶姓名」欄位顯示為空或「未知客戶」
- 訂單詳情頁面中的「司機姓名」欄位顯示為空或「未知司機」

### 問題 3: 司機端訂單詳情路由錯誤

**症狀**:
- 點擊司機端訂單後出現錯誤：`GoException: no routes for location: /driver/order-detail/[訂單ID]`
- 訂單詳情頁面不存在
- 路由配置缺失

---

## 🔍 根本原因

### 問題 1 & 2: API 使用錯誤的查詢方式

**原因**:
1. **使用了 JOIN 語法但沒有使用 service_role key**
   - API 使用 `new DatabaseService()` (anon key)
   - 但查詢需要 JOIN `user_profiles` 表
   - RLS 策略阻止了 anon key 訪問 `user_profiles`

2. **JOIN 語法可能不穩定**
   - 使用 `user_profiles!user_id` 語法
   - 返回的資料結構可能是陣列或物件
   - 導致資料解析失敗

**錯誤代碼**:
```typescript
const db = new DatabaseService(); // ❌ 使用 anon key

let query = db.supabase
  .from('bookings')
  .select(`
    *,
    customer:customer_id (
      id,
      email,
      user_profiles!user_id (first_name, last_name, phone)
    ),
    driver:driver_id (
      id,
      email,
      user_profiles!user_id (first_name, last_name, phone)
    )
  `)
```

### 問題 3: 司機端路由和頁面缺失

**原因**:
1. **訂單詳情頁面不存在**
   - 缺少 `driver_order_detail_page.dart`

2. **路由配置缺失**
   - `driver_router.dart` 中沒有 `/driver/order-detail/:orderId` 路由

3. **導航代碼已存在**
   - `driver_order_page.dart` 中已經有導航代碼：`context.push('/driver/order-detail/${order.id}')`
   - 但目標路由不存在

## 🔧 修復方案

### 修復 1: 公司端訂單列表 API

**文件**: `web-admin/src/app/api/admin/bookings/route.ts`

**修改內容**:
1. 使用 `DatabaseService(true)` 啟用 service_role key
2. 改用分離查詢代替 JOIN
3. 手動合併資料

**修改前**:
```typescript
const db = new DatabaseService(); // ❌ anon key

let query = db.supabase
  .from('bookings')
  .select(`
    *,
    customer:customer_id (...),
    driver:driver_id (...)
  `)
```

**修改後**:
```typescript
const db = new DatabaseService(true); // ✅ service_role key

// 1. 獲取訂單基本資料
let query = db.supabase
  .from('bookings')
  .select('*', { count: 'exact' })
  .order('created_at', { ascending: false });

// 2. 獲取所有相關用戶 ID
const customerIds = [...new Set(bookings?.map(b => b.customer_id).filter(Boolean))];
const driverIds = [...new Set(bookings?.map(b => b.driver_id).filter(Boolean))];
const allUserIds = [...new Set([...customerIds, ...driverIds])];

// 3. 分別查詢 profiles、drivers、users
const { data: profiles } = await db.supabase
  .from('user_profiles')
  .select('*')
  .in('user_id', allUserIds);

const { data: driverInfos } = await db.supabase
  .from('drivers')
  .select('*')
  .in('user_id', driverIds);

const { data: users } = await db.supabase
  .from('users')
  .select('id, email, firebase_uid')
  .in('id', allUserIds);

// 4. 創建映射並合併資料
const profileMap = new Map();
profiles?.forEach(p => profileMap.set(p.user_id, p));

const driverInfoMap = new Map();
driverInfos?.forEach(d => driverInfoMap.set(d.user_id, d));

const userMap = new Map();
users?.forEach(u => userMap.set(u.id, u));

// 5. 格式化訂單資料
const formattedBookings = (bookings || []).map(booking => {
  const customer = userMap.get(booking.customer_id);
  const customerProfile = profileMap.get(booking.customer_id);
  
  const driver = booking.driver_id ? userMap.get(booking.driver_id) : null;
  const driverProfile = booking.driver_id ? profileMap.get(booking.driver_id) : null;
  const driverInfo = booking.driver_id ? driverInfoMap.get(booking.driver_id) : null;

  return {
    // ... 格式化資料
    customer: {
      id: customer?.id,
      name: customerProfile 
        ? `${customerProfile.first_name || ''} ${customerProfile.last_name || ''}`.trim() || '未知客戶'
        : '未知客戶',
      phone: customerProfile?.phone || '無電話',
      email: customer?.email,
    },
    driver: driver ? {
      id: driver.id,
      name: driverProfile 
        ? `${driverProfile.first_name || ''} ${driverProfile.last_name || ''}`.trim() || '未知司機'
        : '未知司機',
      phone: driverProfile?.phone || '無電話',
      email: driver.email,
      vehicleType: driverInfo?.vehicle_type,
      vehiclePlate: driverInfo?.vehicle_plate,
    } : null,
  };
});
```

**關鍵代碼**:
```dart
class DriverOrderDetailPage extends ConsumerWidget {
  final String orderId;

  const DriverOrderDetailPage({
    super.key,
    required this.orderId,
  });

  @override
  Widget build(BuildContext context, WidgetRef ref) {
    final orderAsync = ref.watch(bookingProvider(orderId));

    return Scaffold(
      appBar: AppBar(
        title: const Text('訂單詳情'),
        backgroundColor: const Color(0xFF4CAF50),
        foregroundColor: Colors.white,
      ),
      body: orderAsync.when(
        data: (order) {
          // ... 顯示訂單詳情
        },
        loading: () => const Center(child: CircularProgressIndicator()),
        error: (error, stack) => Center(child: Text('載入失敗: $error')),
      ),
    );
  }
}
```
