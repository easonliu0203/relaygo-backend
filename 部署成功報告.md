## 🆘 如果遇到問題

### 問題 A：手動觸發返回錯誤

**檢查**：
1. Edge Function 日誌
2. 錯誤訊息
3. 環境變數（FIREBASE_PROJECT_ID, FIREBASE_API_KEY）

**參考**：`supabase/SYNC_FIX_TESTING_GUIDE.md` 故障排除

### 問題 B：processed_at 仍是 NULL

**可能原因**：
1. Firestore 權限問題
2. Firebase API Key 無效
3. 資料格式問題

**診斷**：
1. 查看 Edge Function 日誌
2. 檢查 `error_message` 欄位
3. 執行 `supabase/diagnose-sync-issue.sql`

### 問題 C：App 仍顯示「訂單不存在」

**檢查**：
1. Firestore 是否有文檔
2. 文檔 ID 是否正確
3. 客戶端使用的 ID 是否匹配
4. Firestore 讀取權限
