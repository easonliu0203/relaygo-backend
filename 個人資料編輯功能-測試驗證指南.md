## 🧪 客戶端測試

### 步驟 1: 啟動客戶端應用

```bash
cd d:\repo\mobile
scripts\run-customer.bat
```

**或手動執行**:
```bash
cd d:\repo\mobile
flutter run --flavor customer --target lib/main_customer.dart
```

### 步驟 2: 登入測試帳號

1. 啟動應用後，進入登入頁面
2. 輸入測試帳號：
   - Email: `customer.test@relaygo.com`
   - Password: `RelayGO2024!Customer`
3. 點擊「登入」按鈕
4. 確認成功登入並進入主頁面

### 步驟 3: 進入個人資料頁面

1. 點擊底部導航欄的「個人資料」標籤
2. 確認頁面顯示：
   - 用戶頭像
   - 用戶名稱（可能顯示「客戶用戶」）
   - Email: `customer.test@relaygo.com`
   - 「客戶」標籤

### 步驟 4: 編輯個人資料

1. 點擊「編輯個人資料」選項
2. 確認進入編輯頁面
3. 填寫以下資訊：

   **基本資訊**:
   - 姓氏: `測試`
   - 名字: `客戶`
   - 電話號碼: `0912345678`
   - 性別: 選擇「男性」
   - 生日: 選擇 `1990-01-01`
   - 地址: `台北市信義區信義路五段7號`

   **緊急聯絡人**:
   - 緊急聯絡人姓名: `測試家人`
   - 緊急聯絡人電話: `0911111111`

4. 點擊右上角的「✓」按鈕保存

### 步驟 5: 驗證保存結果

1. 確認顯示綠色的「個人資料已更新」訊息
2. 自動返回個人資料頁面
3. 確認頁面顯示：
   - 用戶名稱: `測試客戶`
   - 電話: `0912345678`

### 步驟 6: 測試資料持久化

1. 關閉應用（完全退出）
2. 重新啟動應用
3. 登入相同的測試帳號
4. 進入個人資料頁面
5. 點擊「編輯個人資料」
6. 確認所有填寫的資料都還在

### 步驟 7: 測試表單驗證

1. 進入編輯頁面
2. 清空「姓氏」欄位
3. 點擊保存
4. 確認顯示「請輸入姓氏」錯誤訊息

5. 填寫姓氏，清空「名字」欄位
6. 點擊保存
7. 確認顯示「請輸入名字」錯誤訊息

8. 填寫名字，清空「電話號碼」欄位
9. 點擊保存
10. 確認顯示「請輸入電話號碼」錯誤訊息

11. 填寫電話號碼為 `123`（不足10位）
12. 點擊保存
13. 確認顯示「請輸入有效的電話號碼（10位數字）」錯誤訊息

## 🔍 故障排除

### 問題 1: 編譯錯誤 - 找不到 Freezed 生成的文件

**症狀**:
```
Error: Couldn't resolve the import 'user_profile.freezed.dart'
```

**解決方案**:
```bash
cd d:\repo\mobile
flutter packages pub run build_runner build --delete-conflicting-outputs
```

### 問題 2: Supabase 連接失敗

**症狀**:
```
SupabaseServiceException: 獲取用戶資料失敗
```

**檢查**:
1. 確認 `.env` 文件中的 `SUPABASE_URL` 和 `SUPABASE_ANON_KEY` 正確
2. 確認 Supabase 專案正常運行
3. 檢查網路連接

**解決方案**:
```bash
# 重新啟動應用
flutter run --flavor customer --target lib/main_customer.dart
```

### 問題 3: 保存失敗 - 用戶未登入

**症狀**:
```
更新失敗：Exception: 用戶未登入
```

**解決方案**:
1. 確認已成功登入
2. 檢查 Firebase Auth 狀態
3. 重新登入

### 問題 4: 電話號碼驗證失敗

**症狀**:
```
請輸入有效的電話號碼（10位數字）
```

**解決方案**:
- 確保電話號碼是10位數字
- 不要包含空格或特殊字符
- 例如: `0912345678`

### 問題 5: 資料沒有顯示在公司端

**症狀**: 公司端管理頁面看不到更新的資料

**檢查**:
1. 確認資料已成功保存（檢查 Supabase 控制台）
2. 重新整理公司端頁面
3. 清除瀏覽器快取

**解決方案**:
```sql
-- 在 Supabase SQL Editor 中執行
SELECT * FROM user_profiles WHERE user_id IN (
  SELECT id FROM users WHERE email IN ('customer.test@relaygo.com', 'driver.test@relaygo.com')
);
```
