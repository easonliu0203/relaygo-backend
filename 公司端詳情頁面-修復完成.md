## 💡 技術亮點

### 1. 動態路由

使用 Next.js 的動態路由功能：
- `[id]` 目錄表示動態參數
- 使用 `useParams()` 獲取 ID
- 支持任意 ID 的詳情頁面

### 2. 統計資料計算

**客戶 VIP 等級**:
```typescript
let vipLevel = 'bronze';
if (totalSpent >= 100000) vipLevel = 'diamond';
else if (totalSpent >= 50000) vipLevel = 'platinum';
else if (totalSpent >= 20000) vipLevel = 'gold';
else if (totalSpent >= 5000) vipLevel = 'silver';
```

**司機統計**:
- 從訂單表查詢司機的所有訂單
- 計算總行程數、完成行程數
- 計算總收入（只計算已完成訂單）

### 3. 資料關聯查詢

使用 Supabase 的分離查詢 + 手動合併：
```typescript
// 1. 查詢用戶基本資訊
const { data: user } = await db.supabase
  .from('users')
  .select('*')
  .eq('id', driverId)
  .single();

// 2. 查詢個人資料
const { data: profile } = await db.supabase
  .from('user_profiles')
  .select('*')
  .eq('user_id', driverId)
  .single();

// 3. 查詢司機專屬資料
const { data: driverInfo } = await db.supabase
  .from('drivers')
  .select('*')
  .eq('user_id', driverId)
  .single();

// 4. 合併資料
const formattedDriver = {
  ...user,
  ...profile,
  ...driverInfo,
};
```

### 4. UI 組件

使用 Ant Design 組件：
- `Descriptions`: 顯示詳細資訊
- `Statistic`: 顯示統計卡片
- `Table`: 顯示最近訂單
- `Tag`: 顯示狀態標籤
- `Avatar`: 顯示頭像
