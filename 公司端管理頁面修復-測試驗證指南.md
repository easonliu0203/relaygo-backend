### 問題 2: 測試帳號不顯示

**症狀**: 搜尋測試帳號時找不到

**檢查**:
1. 確認測試帳號存在於資料庫
2. 確認 `role` 欄位正確
3. 確認 API 查詢條件正確

**解決方案**:
```sql
-- 檢查測試帳號
SELECT * FROM users WHERE email IN ('customer.test@relaygo.com', 'driver.test@relaygo.com');

-- 檢查 user_profiles
SELECT up.*, u.email, u.role
FROM user_profiles up
JOIN users u ON up.user_id = u.id
WHERE u.email IN ('customer.test@relaygo.com', 'driver.test@relaygo.com');

-- 檢查 drivers
SELECT d.*, u.email
FROM drivers d
JOIN users u ON d.user_id = u.id
WHERE u.email = 'driver.test@relaygo.com';
```

### 問題 4: 資料不完整

**症狀**: 測試帳號顯示但資料不完整（姓名、電話等缺失）

**檢查**:
1. 確認 `user_profiles` 表有對應記錄
2. 確認 `drivers` 表有對應記錄（司機）
3. 確認資料欄位有值

**解決方案**:
```sql
-- 為測試客戶添加 user_profile
INSERT INTO user_profiles (user_id, first_name, last_name, phone)
SELECT id, '測試', '客戶', '0912345678'
FROM users
WHERE email = 'customer.test@relaygo.com'
AND NOT EXISTS (
  SELECT 1 FROM user_profiles WHERE user_id = users.id
);

-- 為測試司機添加 user_profile
INSERT INTO user_profiles (user_id, first_name, last_name, phone)
SELECT id, '測試', '司機', '0987654321'
FROM users
WHERE email = 'driver.test@relaygo.com'
AND NOT EXISTS (
  SELECT 1 FROM user_profiles WHERE user_id = users.id
);

-- 為測試司機添加 drivers 記錄
INSERT INTO drivers (user_id, first_name, last_name, phone, vehicle_type, vehicle_plate, vehicle_model, is_available, status)
SELECT id, '測試', '司機', '0987654321', 'A', 'TEST-001', 'Toyota Alphard', true, 'active'
FROM users
WHERE email = 'driver.test@relaygo.com'
AND NOT EXISTS (
  SELECT 1 FROM drivers WHERE user_id = users.id
);
```
