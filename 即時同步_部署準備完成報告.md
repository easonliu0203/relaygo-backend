## ⚠️ 常見問題和解決方法

### 問題 1: pg_net 擴展未啟用

**症狀**: Migration 執行時報錯 "extension pg_net does not exist"

**解決方法**:
1. 前往 Extensions 頁面
2. 搜尋 "pg_net"
3. 點擊 "Enable"
4. 重新執行 Migration

### 問題 2: Service Role Key 無法獲取

**症狀**: Trigger 執行時警告 "Unable to get service_role_key"

**解決方法**:
- 這是正常的警告，不影響功能
- Trigger 會繼續執行，Cron Job 會補償

### 問題 3: HTTP 請求失敗

**症狀**: net._http_response 顯示 status_code 非 200

**解決方法**:
1. 檢查 Edge Function 是否正常運行
2. 查看 Edge Function 日誌
3. 確認 URL 正確
4. 驗證 Authorization header
