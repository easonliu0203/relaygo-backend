### 錯誤 5: status 值違反 CHECK 約束 ✅
```
ERROR: 23514: new row for relation "bookings" violates check constraint "bookings_status_check"
```
**修復**: 將 `status: 'pending'` 改為 `status: 'draft'`

**原因**: 實際數據庫使用的 status 約束為：
```sql
CHECK (status IN (
  'draft', 'pending_payment', 'paid_deposit', 'assigned', 'driver_confirmed',
  'driver_departed', 'driver_arrived', 'trip_started', 'trip_ended',
  'pending_balance', 'completed', 'cancelled', 'refunded'
))
```
