## 📋 問題描述

### 問題現象

**受影響的頁面**:
1. 公司端訂單管理頁面（http://localhost:3001/orders）
2. 待處理訂單頁面（http://localhost:3001/orders/pending）
3. 進行中訂單頁面（http://localhost:3001/orders/confirmed）
4. 已完成訂單頁面（http://localhost:3001/orders/completed）

**當前問題**:
1. **缺少「建立時間」欄位**:
   - 訂單列表中只顯示「預約時間」
   - 沒有顯示「建立時間」（created_at）欄位
   - 無法知道訂單是何時創建的

2. **預約時間顯示錯誤**:
   - 預約時間欄位顯示 "Invalid Date"
   - 看不到正確的日期和時間數字
   - 可能是日期格式解析錯誤

**預期行為**:
1. 訂單列表應該同時顯示：
   - 建立時間（created_at）- 訂單創建的時間
   - 預約時間（start_date + start_time）- 客戶預約的用車時間

2. 所有時間欄位應該正確顯示：
   - 格式：YYYY-MM-DD HH:mm
   - 例如：2025-10-09 14:30
   - 不應該顯示 "Invalid Date"

## 🔍 問題診斷過程

### 1. 檢查訂單列表頁面

**文件**: `web-admin/src/app/orders/page.tsx`

**問題代碼** (第 170-174 行):
```typescript
{
  title: '預約時間',
  dataIndex: 'scheduledTime',  // ❌ 錯誤：這個欄位不存在
  key: 'scheduledTime',
  render: (time: string) => dayjs(time).format('MM/DD HH:mm'),  // ❌ time 是 undefined
},
```

**問題分析**:
- 使用 `dataIndex: 'scheduledTime'` 嘗試獲取單一欄位
- 但 API 返回的是 `scheduledDate` 和 `scheduledTime` 兩個欄位
- `scheduledTime` 欄位不存在，導致 `time` 為 `undefined`
- `dayjs(undefined)` 返回 "Invalid Date"

### 2. 檢查 API 返回的資料格式

**文件**: `web-admin/src/app/api/admin/bookings/route.ts`

**API 返回的資料** (第 108-157 行):
```typescript
return {
  id: booking.id,
  bookingNumber: booking.booking_number,
  status: booking.status,
  
  // 訂單詳情
  vehicleType: booking.vehicle_type,
  pickupLocation: booking.pickup_location,
  dropoffLocation: booking.destination,
  scheduledDate: booking.start_date,      // ✅ 有返回
  scheduledTime: booking.start_time,      // ✅ 有返回
  durationHours: booking.duration_hours,
  
  // 時間戳
  createdAt: booking.created_at,          // ✅ 有返回
  updatedAt: booking.updated_at,
  
  ...
};
```

**發現**:
- ✅ API 返回了 `createdAt` 欄位
- ✅ API 返回了 `scheduledDate` 和 `scheduledTime` 欄位
- ❌ 前端沒有顯示 `createdAt`
- ❌ 前端沒有正確組合 `scheduledDate` 和 `scheduledTime`

### 3. 根本原因分析

**問題 1**: 缺少「建立時間」欄位
- API 返回了 `createdAt`
- 前端沒有在表格中顯示這個欄位

**問題 2**: 預約時間顯示 "Invalid Date"
- API 返回的是 `scheduledDate` 和 `scheduledTime` 兩個欄位
- 前端使用 `dataIndex: 'scheduledTime'` 嘗試獲取單一欄位
- 這個欄位名稱與 API 返回的不匹配
- 需要組合 `scheduledDate` 和 `scheduledTime` 來顯示完整的預約時間

**錯誤流程**:
```
API 返回: { scheduledDate: '2025-10-09', scheduledTime: '14:30' }
  ↓
前端嘗試獲取: record.scheduledTime
  ↓
結果: undefined (因為沒有這個欄位)
  ↓
dayjs(undefined).format('MM/DD HH:mm')
  ↓
顯示: "Invalid Date"
```

**正確流程**:
```
API 返回: { scheduledDate: '2025-10-09', scheduledTime: '14:30' }
  ↓
前端組合: `${record.scheduledDate} ${record.scheduledTime}`
  ↓
結果: '2025-10-09 14:30'
  ↓
dayjs('2025-10-09 14:30').format('YYYY-MM-DD HH:mm')
  ↓
顯示: "2025-10-09 14:30"
```

### 方案概述

1. **添加「建立時間」欄位**

2. **修復「預約時間」欄位**
   - 組合 `scheduledDate` 和 `scheduledTime`
   - 正確格式化日期時間
   - 添加排序功能

3. **創建日期格式化函數**
   - 統一的日期格式化邏輯
   - 錯誤處理
   - 支持只有日期或日期+時間

4. **優化表格欄位**
   - 添加欄位寬度
   - 添加安全檢查（?.）
   - 固定操作欄位

### 修復步驟

#### 步驟 1: 創建日期格式化函數

**所有訂單頁面都添加此函數**:

```typescript
// 格式化日期時間
const formatDateTime = (date: string, time?: string) => {
  if (!date) return '-';
  try {
    if (time) {
      // 組合日期和時間
      const dateTimeStr = `${date} ${time}`;
      return dayjs(dateTimeStr).format('YYYY-MM-DD HH:mm');
    }
    // 只有日期
    return dayjs(date).format('YYYY-MM-DD');
  } catch (error) {
    console.error('日期格式化錯誤:', error);
    return '-';
  }
};
```

**功能**:
- 接受 `date` 和可選的 `time` 參數
- 如果有 `time`，組合日期和時間
- 如果只有 `date`，只格式化日期
- 錯誤處理，返回 `-` 而不是 "Invalid Date"

#### 步驟 2: 添加「建立時間」欄位

**新增欄位**:

```typescript
{
  title: '建立時間',
  dataIndex: 'createdAt',
  key: 'createdAt',
  width: 150,
  sorter: (a: any, b: any) => dayjs(a.createdAt).unix() - dayjs(b.createdAt).unix(),
  render: (createdAt: string) => {
    if (!createdAt) return '-';
    try {
      return dayjs(createdAt).format('YYYY-MM-DD HH:mm');
    } catch (error) {
      console.error('建立時間格式化錯誤:', error);
      return '-';
    }
  },
},
```

**功能**:
- 顯示訂單建立時間
- 支持排序（點擊欄位標題）
- 錯誤處理
- 固定寬度 150px

#### 步驟 3: 修復「預約時間」欄位

**修改前**:
```typescript
{
  title: '預約時間',
  dataIndex: 'scheduledTime',  // ❌ 錯誤
  key: 'scheduledTime',
  render: (time: string) => dayjs(time).format('MM/DD HH:mm'),  // ❌ time 是 undefined
},
```

**修改後**:
```typescript
{
  title: '預約時間',
  key: 'scheduledDateTime',
  width: 150,
  sorter: (a: any, b: any) => {
    const dateA = `${a.scheduledDate} ${a.scheduledTime || '00:00'}`;
    const dateB = `${b.scheduledDate} ${b.scheduledTime || '00:00'}`;
    return dayjs(dateA).unix() - dayjs(dateB).unix();
  },
  render: (_, record: any) => formatDateTime(record.scheduledDate, record.scheduledTime),
},
```

**功能**:
- 組合 `scheduledDate` 和 `scheduledTime`
- 使用 `formatDateTime` 函數格式化
- 支持排序
- 固定寬度 150px

#### 步驟 4: 優化其他欄位

// utils/dateFormatter.ts
import dayjs from 'dayjs';

export const formatDateTime = (date: string, time?: string): string => {
  if (!date) return '-';
  try {
    if (time) {
      const dateTimeStr = `${date} ${time}`;
      return dayjs(dateTimeStr).format('YYYY-MM-DD HH:mm');
    }
    return dayjs(date).format('YYYY-MM-DD');
  } catch (error) {
    console.error('日期格式化錯誤:', error);
    return '-';
  }
};

export const formatDate = (date: string): string => {
  if (!date) return '-';
  try {
    return dayjs(date).format('YYYY-MM-DD');
  } catch (error) {
    console.error('日期格式化錯誤:', error);
    return '-';
  }
};

export const formatTime = (time: string): string => {
  if (!time) return '-';
  try {
    return dayjs(time, 'HH:mm').format('HH:mm');
  } catch (error) {
    console.error('時間格式化錯誤:', error);
    return '-';
  }
};
