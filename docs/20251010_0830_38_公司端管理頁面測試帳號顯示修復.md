## 🔍 問題診斷

### 診斷步驟

1. **檢查頁面代碼** ✅
   - 客戶管理頁面：`web-admin/src/app/customers/page.tsx`
   - 司機管理頁面：`web-admin/src/app/drivers/page.tsx`

2. **發現根本原因** ❌
   - **客戶管理頁面使用模擬數據**（`mockCustomers`）
   - **司機管理頁面使用模擬數據**（`mockDrivers`）
   - **沒有從 Supabase 查詢真實數據**

3. **檢查 API 端點** ⚠️
   - 客戶管理 API 不存在：`/api/admin/customers`
   - 司機管理 API 不存在：`/api/admin/drivers`
   - 只有可用司機 API：`/api/admin/drivers/available`

### 根本原因

**公司端管理頁面使用模擬數據，沒有實作真實的 API 查詢**：

| 頁面 | 數據來源 | 問題 |
|------|----------|------|
| 客戶管理 | `mockCustomers` | ❌ 模擬數據，無法顯示真實客戶 |
| 司機管理 | `mockDrivers` | ❌ 模擬數據，無法顯示真實司機 |

### 2. 創建司機管理 API

**文件**: `web-admin/src/app/api/admin/drivers/route.ts`

**功能**:
- 從 Supabase 查詢司機列表
- 支持狀態篩選（`status`）
- 支持車型篩選（`vehicleType`）
- 支持搜尋功能（`search`）
- 支持分頁（`limit`, `offset`）
- JOIN `user_profiles` 和 `drivers` 表獲取完整資料

**關鍵代碼**:
```typescript
let query = db.supabase
  .from('users')
  .select(`
    *,
    user_profiles!user_id (
      first_name,
      last_name,
      phone,
      avatar_url
    ),
    drivers!user_id (
      id,
      first_name,
      last_name,
      phone,
      license_number,
      vehicle_type,
      vehicle_plate,
      vehicle_model,
      is_available,
      status,
      rating,
      total_trips,
      created_at
    )
  `, { count: 'exact' })
  .eq('role', 'driver')
  .order('created_at', { ascending: false });

// 格式化司機資料
const formattedDrivers = filteredDrivers.map(driver => {
  const profile = driver.user_profiles?.[0];
  const driverInfo = driver.drivers?.[0];
  
  // 優先使用 drivers 表的資料
  const firstName = driverInfo?.first_name || profile?.first_name || '';
  const lastName = driverInfo?.last_name || profile?.last_name || '';
  
  return {
    id: driver.id,
    firebase_uid: driver.firebase_uid,
    email: driver.email,
    name: `${firstName} ${lastName}`.trim() || '未設定',
    // 司機特定資訊
    licenseNumber: driverInfo?.license_number || '',
    vehicleType: driverInfo?.vehicle_type || '',
    vehiclePlate: driverInfo?.vehicle_plate || '',
    rating: driverInfo?.rating || 0,
    totalTrips: driverInfo?.total_trips || 0,
    // ...
  };
});
```

### 3. 修改客戶管理頁面


3. **添加載入函數**:
```typescript
const loadCustomers = async () => {
  setLoading(true);
  try {
    const params: any = {
      limit: 100,
      offset: 0,
    };

    if (statusFilter && statusFilter !== 'all') {
      params.status = statusFilter;
    }

    if (searchText) {
      params.search = searchText;
    }

    const response = await ApiService.getCustomers(params);

    if (response.success) {
      setCustomers(response.data || []);
      setTotal(response.total || 0);
    }
  } catch (error) {
    message.error('載入客戶失敗');
  } finally {
    setLoading(false);
  }
};
```

4. **添加初始載入和篩選監聽**:
```typescript

  loadCustomers();
}, []);

useEffect(() => {
  loadCustomers();
}, [statusFilter]);
```
