```typescript

  provider: PaymentProviderType.MOCK, // 或 OFFLINE
  isTestMode: true,
  config: {
    successRate: 0.9, // 90% 成功率
    processingDelay: 2000 // 2秒處理延遲
  }
};
```

## 風險控制

### 技術風險
- **中等**: 模擬支付可能無法完全模擬真實場景
- **解決方案**: 詳細記錄支付流程，為後續整合做準備

### 業務風險
- **低**: 封測用戶可能對模擬支付感到困惑
- **解決方案**: 清楚標示測試環境，提供詳細說明

### 時程風險
- **中等**: 功能簡化可能影響用戶體驗測試
- **解決方案**: 重點測試核心流程，收集關鍵回饋
