## 4. 手動測試

### 4.1 功能測試

**測試環境**: Android/iOS 實體裝置或模擬器

**測試案例**:

1. **語言精靈測試**
   - [ ] 首次登入時顯示語言精靈
   - [ ] 選擇不同語言並保存
   - [ ] 驗證語言偏好已保存到 Firestore
   - [ ] 再次登入時不顯示語言精靈

2. **語言設定測試**
   - [ ] 在設定頁面修改顯示語言（preferredLang）
   - [ ] 在設定頁面修改輸入語言（inputLangHint）
   - [ ] 驗證設定已保存到 Firestore

3. **聊天室語言切換測試**
   - [ ] 點擊地球按鈕顯示語言選擇 Bottom Sheet
   - [ ] 選擇不同語言
   - [ ] 驗證訊息重新翻譯
   - [ ] 驗證語言選擇已保存到 SharedPreferences

4. **訊息翻譯測試**
   - [ ] 發送中文訊息，驗證 `detectedLang` 為 `zh-TW`
   - [ ] 切換到英文顯示，驗證訊息被翻譯為英文
   - [ ] 點擊「顯示原文」按鈕，驗證顯示原文
   - [ ] 點擊「顯示翻譯」按鈕，驗證顯示翻譯

5. **快取測試**
   - [ ] 第一次翻譯訊息，觀察載入指示器
   - [ ] 切換到其他語言再切換回來，驗證快取命中（無載入指示器）
   - [ ] 清除應用資料，驗證快取被清除

6. **語言優先順序測試**
   - [ ] 設定全域語言為英文
   - [ ] 在聊天室設定語言為日文
   - [ ] 驗證聊天室顯示日文（roomViewLang > preferredLang）
   - [ ] 移除聊天室語言設定
   - [ ] 驗證聊天室顯示英文（preferredLang）

### 4.2 效能測試

**測試案例**:

1. **翻譯速度測試**
   - [ ] 測試單條訊息翻譯時間（目標：< 2 秒）
   - [ ] 測試批次翻譯時間（10 條訊息，目標：< 5 秒）
   - [ ] 測試快取命中時間（目標：< 100ms）

2. **記憶體使用測試**
   - [ ] 測試快取大小（100 條訊息，目標：< 5MB）
   - [ ] 測試長時間使用後的記憶體洩漏

3. **網路使用測試**
   - [ ] 測試翻譯 API 調用次數（目標：減少 90%）
   - [ ] 測試網路流量（目標：< 1MB/100 條訊息）

### 4.3 錯誤處理測試

**測試案例**:

1. **網路錯誤測試**
   - [ ] 關閉網路，驗證顯示原文
   - [ ] 恢復網路，驗證自動重試翻譯

2. **API 錯誤測試**
   - [ ] 模擬 429 錯誤（速率限制），驗證錯誤訊息
   - [ ] 模擬 401 錯誤（未授權），驗證錯誤訊息

3. **快取錯誤測試**
   - [ ] 刪除快取檔案，驗證應用不崩潰
   - [ ] 損壞快取檔案，驗證應用自動清理
