## 🔧 技術實作

### 1. 使用 ConsumerStatefulWidget

**原因**: 需要在 Shell 組件中使用 Riverpod 的 `ref.watch()` 來監聽未讀訊息總數。

**修改前**:
```dart
class CustomerShell extends StatefulWidget {
  // ...
}

class _CustomerShellState extends State<CustomerShell> {
  // ...
}
```

**修改後**:
```dart
class CustomerShell extends ConsumerStatefulWidget {
  // ...
}

class _CustomerShellState extends ConsumerState<CustomerShell> {
  // ...
}
```

### 2. 使用 StreamProvider 監聽未讀訊息

**Provider**:
```dart
final totalUnreadCountStreamProvider = StreamProvider<int>((ref) {
  final chatService = ref.watch(chatServiceProvider);
  return chatService.getTotalUnreadCountStream();
});
```

**使用**:
```dart
final totalUnreadCountAsync = ref.watch(totalUnreadCountStreamProvider);
```

**優點**:
- 自動處理載入和錯誤狀態
- 即時更新未讀訊息數量
- 無需手動管理訂閱

### 3. 使用 Badge 組件顯示未讀數量

**Flutter Badge 組件**:
```dart
Badge(
  label: Text(count > 99 ? '99+' : count.toString()),
  child: const Icon(Icons.chat),
)
```

**顯示邏輯**:
- 未讀數量 = 0：不顯示徽章
- 未讀數量 1-99：顯示具體數字
- 未讀數量 > 99：顯示「99+」

**樣式**:
- 紅色背景
- 白色文字
- 位於圖標右上角

### 4. 處理異步狀態

**使用 AsyncValue.when()**:
```dart
totalUnreadCountAsync.when(
  data: (count) => /* 顯示徽章 */,
  loading: () => /* 顯示圖標 */,
  error: (_, __) => /* 顯示圖標 */,
)
```

**優點**:
- 自動處理載入狀態
- 自動處理錯誤狀態
- 代碼簡潔明瞭

## 💡 開發心得和經驗總結

### 成功的地方

1. **無縫整合**:
   - 使用現有的 GoRouter 和 ShellRoute 架構
   - 不需要修改底部導覽列的基本結構
   - 只需替換頁面和添加徽章邏輯

2. **即時更新**:
   - 使用 StreamProvider 實現未讀訊息數量的即時更新
   - 無需手動刷新或輪詢
   - 用戶體驗流暢

3. **代碼重用**:
   - 客戶端和司機端使用相同的 `ChatListPage` 組件
   - 使用相同的 `totalUnreadCountStreamProvider`
   - 減少代碼重複

4. **錯誤處理**:
   - 使用 `AsyncValue.when()` 自動處理載入和錯誤狀態
   - 即使載入失敗，也不會影響底部導覽列的顯示

### 遇到的困難

1. **StatefulWidget vs ConsumerStatefulWidget**:
   - **問題**: 原本的 Shell 組件是 `StatefulWidget`，無法使用 `ref.watch()`
   - **解決**: 改為 `ConsumerStatefulWidget` 和 `ConsumerState`

2. **Badge 組件的使用**:
   - **問題**: 需要根據未讀數量動態顯示或隱藏徽章
   - **解決**: 使用三元運算符判斷是否顯示 Badge

3. **異步狀態處理**:
   - **問題**: StreamProvider 返回的是 `AsyncValue`，需要處理載入和錯誤狀態
   - **解決**: 使用 `AsyncValue.when()` 方法

### 最佳實踐

1. **使用 ConsumerStatefulWidget**:
   ```dart
   // ✅ 正確
   class MyShell extends ConsumerStatefulWidget {
     @override
     ConsumerState<MyShell> createState() => _MyShellState();
   }
   
   class _MyShellState extends ConsumerState<MyShell> {
     @override
     Widget build(BuildContext context) {
       final data = ref.watch(someProvider);
       // ...
     }
   }
   ```

2. **處理異步狀態**:
   ```dart
   // ✅ 正確
   asyncValue.when(
     data: (data) => /* 顯示數據 */,
     loading: () => /* 顯示載入 */,
     error: (error, stack) => /* 顯示錯誤 */,
   )
   ```

3. **條件顯示 Badge**:
   ```dart
   // ✅ 正確
   count > 0
       ? Badge(label: Text('$count'), child: Icon(Icons.chat))
       : Icon(Icons.chat)
   ```
