## 🔧 關鍵轉換點

### 轉換點 1：欄位映射

```
Supabase 欄位          →  Firestore 欄位
─────────────────────────────────────────
destination            →  dropoffAddress
startDate + startTime  →  bookingTime
total_amount           →  estimatedFare
special_requirements   →  notes
```

---

### 轉換點 2：GeoPoint 格式

```
Supabase payload:
{ latitude: 25.033, longitude: 121.5654 }
    ↓
Edge Function 標記:
{ _latitude: 25.033, _longitude: 121.5654 }
    ↓
convertToFirestoreFields 轉換:
{ geoPointValue: { latitude: 25.033, longitude: 121.5654 } }
    ↓
Firestore 存儲:
geopoint (25.033, 121.5654)
    ↓
客戶端解析:
LocationPoint(25.033, 121.5654)
```

---

### 轉換點 3：Timestamp 格式

```
Supabase payload:
"2025-10-06T05:12:00"
    ↓
Edge Function 標記:
{ _timestamp: "2025-10-06T05:12:00" }
    ↓
convertToFirestoreFields 轉換:
{ timestampValue: "2025-10-06T05:12:00.000Z" }
    ↓
Firestore 存儲:
timestamp (October 6, 2025 at 5:12:00 AM UTC+8)
    ↓
客戶端解析:
DateTime(2025, 10, 6, 5, 12)
```
