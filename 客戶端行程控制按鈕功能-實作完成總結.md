## 🔄 狀態流程

### Supabase 狀態流程

```
pending_payment → paid_deposit → assigned → matched → 
driver_confirmed → driver_departed → driver_arrived → 
trip_started ⭐ → trip_ended ⭐ → pending_balance ⭐ → completed
```

### Firestore 狀態流程

```
pending → pending → pending → pending → 
matched → inProgress → inProgress → 
inProgress ⭐ → completed ⭐ → completed ⭐ → completed
```

### 客戶端按鈕顯示

```
matched/inProgress → 顯示「開始行程」按鈕
inProgress (trip_started) → 顯示「結束行程」按鈕
completed (trip_ended) → 顯示「支付尾款」按鈕（待實作）
completed → 不顯示操作按鈕
```
