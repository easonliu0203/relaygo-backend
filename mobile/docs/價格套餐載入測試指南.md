# 價格套餐載入測試指南

## 🎯 測試目的

驗證價格套餐載入功能在各種情況下都能正常工作，包括：
- 後端服務正常運行
- 後端服務未啟動
- 網路連接錯誤
- API 回應異常

## 📋 測試前準備

### 1. 確認代碼已更新

檢查 `mobile/lib/core/services/pricing_service.dart` 是否包含以下修改：

```dart
// 檢查 import
import 'dart:async';
import 'dart:io';

// 檢查超時處理
final response = await http.get(
  Uri.parse('$_baseUrl/pricing/packages'),
  headers: {'Content-Type': 'application/json'},
).timeout(
  const Duration(seconds: 5),
  onTimeout: () {
    debugPrint('[PricingService] API 請求超時，使用模擬資料');
    throw TimeoutException('API 請求超時');
  },
);
```

### 2. 重新建置應用

```bash
cd mobile
flutter clean
flutter pub get
flutter run --flavor customer --target lib/apps/customer/main_customer.dart
```

## 🐛 常見問題排除

### 問題 1：仍然長時間載入

**可能原因**：
- 代碼沒有重新編譯
- 使用了舊的 APK

**解決方法**：
```bash
flutter clean
flutter pub get
flutter run --flavor customer --target lib/apps/customer/main_customer.dart
```

### 問題 2：沒有看到日誌

**可能原因**：
- 日誌被過濾
- 使用 Release 模式

**解決方法**：
- 確保使用 Debug 模式
- 在 IDE 中查看 Debug Console
- 或使用 `flutter logs` 命令

### 問題 3：模擬資料沒有顯示

**可能原因**：
- `_getMockPackages()` 方法有問題
- 資料格式不匹配

**解決方法**：
- 檢查 `VehiclePackage.fromJson()` 方法
- 確認模擬資料格式正確
